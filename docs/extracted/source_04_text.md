拾光源点·先锋社区智能体项目提案 
 
一、项目概述与未来愿景 
• 项目名称与定位：拾光源点 — 从线下展厅互动屏起步的长期社区记忆系统，定位为“社区
记忆的公民众包平台 + 本地体验入口”。 
• 核心愿景：以沉浸式线下交互与便捷线上访问相结合的方式，收集、验证、保存并展示地
方历史与生活记忆，使这些记忆可被检索、共享与再创造。 
• 理想未来（5 年愿景）： 
o 老街与公共空间恢复人流与活动，展厅成为社区活动节点与教育场所； 
o 跨代沟通常态化：老人讲述、青年数字化整理、儿童参与体验； 
o 居民对本土文化高度自豪，学校将地方史纳入项目式学习； 
o 基于档案催生小型创意经济（导览、手工、纪念品、主题市集）； 
o 系统形成可复制模型，向周边社区扩展并输出方法论与开源工具包。 
 
二、背景与痛点分析 
• 社会背景： 
o 城市化与人口流动导致地方记忆流失； 
o 数字化趋势带来记录机会但缺乏可访问、可信的存证机制； 
o 社区治理、文化振兴与市民参与成为政策与民间关注点。 
文化同质化。现代城市越来越相似，各地的特色正在逐渐消失 
人口老龄化 。大量老年人掌握社区历史，如果不及时记录，可能会导致历史断层，经验
流失 
• 现有痛点： 
o 记忆分散、记载缺乏结构与元数据，难以检索与验证；后果：历史无法系统保存 
o 老人有口述愿望但缺少便捷渠道与信任保障； 
o 年轻人技术能力强但参与缺乏激励与归属感； 
o 线下展陈常制式化、互动性差，难以吸引重复访问； 
o 内容审核与隐私合规机制欠缺，存在伦理与法律风险。 
 
三、经历变革的利益相关者群体 
• 直接交互群体： 
o 社区居民（主动上传/讲述/参观）、学校师生、社区志愿者、游客。 
• 老人（讲述者）：

o 价值：口述史料、地方知识、生活细节； 
o 需求：被尊重的记录流程、简便的参与方式、隐私与知情同意。 
• 儿童（体验者）： 
o 价值：感知地方故事、体验式学习； 
o 需求：沉浸式互动、寓教于乐的内容、家长与学校的安全保障。 
• 年轻群体（共建者）： 
o 价值：内容采集与数字化、技术维护、传播与创意二次创作； 
o 需求：成长与贡献机制（工作坊、志愿学分、微奖励）。 
• 游客（见证者）： 
o 价值：见证与传播、本地文化消费； 
o 需求：易于理解的导览路线、可分享的内容片段。 
• 间接受益/影响群体： 
o 社区组织、地方政府（文化与治理）、学校、文化创意企业、地方经济体（商户、
导览业者）。 
 
四、系统核心模块设计 
• 前端展示与交互（线下 + 线上）： 
o 线下展厅互动屏：多媒体时间轴、地图式记忆探针、触发式故事站点、口述录音/
录像舱； 
o 移动/网页端：可搜索数据库、地图与主题集合、个人/家庭记忆墙、AR 导览与位
置感知内容推送。 
• 多维内容采集： 
o 模态：口述音频、视频访谈、照片、手稿/实物影像、时间/位置元数据、标签与类
别化描述； 
o 采集渠道：现场录制亭、志愿者上门采集、居民自助上传（应用或扫码）、学校项
目采集。 
• 互动与激励机制： 
o 激励：社区贡献积分、志愿者徽章、学校学分/实践证明、主题征集比赛、小额补
贴或展厅优先展出； 
o 互动：故事拼图（多人协作完成一段历史）、代际共创工作坊、月度主题展、“记
忆地图”即时评论与点赞（可匿名）。 
 
五、数据治理与审核机制 
• AI 预处理： 
o 自动转录（语音→文本）、自动分类/标签建议、隐私敏感信息检测（人名、住
址、身份证号等 PII 命中提示）、图像质量/重复检测；

o 生成初步元数据（时间、地点、主题关键词）供人工复核。 
• 管理员复核体系： 
o 多级审核：志愿者初审（事实核对、补充元数据）→ 社区管理员复核（内容合
规、合规签名）→ 专家/长者顾问审阅（历史准确性建议）； 
o 可追溯的审核日志与版本控制，保留原始文件与经编辑的派生版本。 
• 内容与隐私政策： 
o 明确同意机制（录制前签署/电子同意）、可选匿名化、未成年人参与需监护人同
意； 
o 存取权限分级（公开、社区可见、仅管理员可见、仅捐赠者/研究用途），长期保
存与删除流程、版权归属与使用许可（推荐 CC-BY-NC 或社区协定）； 
o 定期审计与申诉流程，明确删除或封存请求的处理时限与责任人。 
 
六、软硬件系统架构 
• 硬件部署： 
o 线下展厅：互动触摸屏（4K 或 1080p，多点触控），独立录音/录像舱（可即时上
传或本地缓存）、本地边缘服务器用于缓存与初步预处理（转码、加密）、UPS 与
防篡改外壳； 
o 可移动采集包：便携录音设备、平板采集站、便携背包式 Wi-Fi（用于上门采
集）。 
• 软件架构： 
o 分层架构：前端（展厅 UI、Web/移动客户端）↔ API 网关 ↔ 后端服务（媒体处
理、转录/AI 模块、检索/索引、用户与权限服务）↔ 数据存储（对象存储：媒体
文件；关系/文档库：元数据与版本；搜索引擎：Elasticsearch/向量检索）； 
o AI/处理：离线与云混合转录与分类服务（隐私敏感项在边缘预处理并打标签后上
传）、NLP 模块用于主题分类与摘要、图像识别用于物品与地点识别； 
o 数据安全：传输加密（TLS）、存储加密、角色基于访问控制（RBAC）、审计日
志、备份与灾备策略； 
o 可扩展性：微服务或容器化部署（Kubernetes），支持多展厅/多社区横向扩展。 
• 运维与本地化： 
o 本地管理员控制台、远程监控仪表板、定期安全与合规检查、开源组件优先以降低
成本与便于社区审计。 
 
七、变革路径与成功指标 
• 变革路径（如何产生影响）： 
o Awareness（提升认知）：通过展厅与学校课程展示地方故事，激发居民对历史的
兴趣；



## Page 4

o Participation（促进参与）：低门槛采集工具与激励机制吸引多年龄层贡献内容；  
o Behaviour（行为转变）：从被动消费到主动讲述与整理，社区自发举办口述会与
主题活动；  
o Relationships（新关系）：代际互动增强，志愿者、学者与居民建立长期协作网
络。 
• 成功的信号（定量与定性） 
o 定量指标： 
▪ 月活跃交互量（线下展厅访问人次、触屏交互次数）； 
▪ 收录档案数与媒体总时长（音频/视频小时数）、分类覆盖度（年代/主题/
地点数量）； 
▪ 参与者分布（年龄层比例）、志愿者与学生项目留存率； 
▪ 内容审核通过率与上链/存证次数（若采用存证方案）。 
o 定性指标： 
▪ 居民的归属感与口述反馈； 
▪ 学校将项目纳入课程或实践案例数； 
▪ 早期用户故事：老居民通过平台重建家庭记忆或引导青年参与的案例； 
▪ 社区活动频次与本地商户参与度提升。 
 
八、风险评估与意外后果 
• 主要风险： 
o 隐私与法律风险：未成年人、敏感信息或未经同意的记录导致纠纷； 
o 排他性：数字鸿沟使部分老人或非网民被排除； 
o 历史争议与偏见：记忆具有主观性，可能引发社区冲突或历史失衡（选择性记
忆）； 
o 误信息/伪造：未充分验证的口述或影像被误用； 
o 维护与资金风险：长期运维成本、硬件损坏或数据遗失。 
• 缓解措施： 
o 严格知情同意、分级权限与快速申诉机制；为不擅长数字的群体提供线下采集与代
理服务； 
o 多源验证流程（交叉核对、时间线核验与专家复核）；建立社区顾问委员会参与争
议裁决； 
o 资金多元化（政府文化基金、企业赞助、众筹、付费体验）与明确维护预算。 
 
九、项目实施与里程碑 
• MVP 阶段（0–9 个月） 
o 目标：完成一个可运转的线下展厅 + 基础采集与展示系统；

o 关键活动： 
▪ 地点选址与社区对接，成立社区顾问小组（含长者代表、学校、街坊、管
理方）； 
▪ 设计并部署 1 台互动触摸屏 + 1 个录音/录像采集舱 + 边缘服务器； 
▪ 开发最小可行软件：媒体上传、转录、基本搜索与展厅 UI；实现同意与权
限流； 
▪ 招募并培训首批志愿者与学生采集队，组织首轮口述采集（目标：200 条
口述或 100 小时媒体）； 
▪ 建立基础数据治理流程与审核角色。 
o 成功里程碑（MVP 退出标准）： 
▪ 展厅连续运营 3 个月，月均访客≥500，人均交互≥2 次； 
▪ 收录媒体≥200 条，转录准确率≥85%，基础权限系统运行稳定。 
• 扩展阶段（9–36 个月） 
o 目标：规模化采集、丰富互动功能、建立可持续运营模式并向周边社区复制； 
o 关键活动： 
▪ 功能迭代：AR 导览、故事拼图协作、移动端优化、AI 辅助标签与摘要；  
▪ 教育合作：与 3–5 所学校建立课程化采集项目；设置学生实践学分机制；  
▪ 社区活动化：主题展、口述节、创意市集、居民故事竞赛；  
▪ 治理与合规：完善长期保存策略、法律顾问审查、开放数据策略（视权
限）；  
▪ 资金与商业化：争取文化专项资金、社企合作、付费导览与捐赠机制；  
▪ 横向复制：输出“展厅部署包”与操作手册，试点 2–3 个邻近社区。 
o 成功里程碑（扩展标准）： 
▪ 平台累计档案≥5,000 条，覆盖年龄分布与主题多样性；  
▪ 学校合作项目常态化，志愿者年留存率≥60%；  
▪ 展厅网络化：至少 3 个展厅稳定运行并形成资料互通；  
▪ 出现实质性社会反馈：本地活动增多、本土品牌或创意商品出现、居民归
属感提升的定性证据。 
• 长期（36–60 个月） 
o 标准化模板与开源工具包、区域文化生态链初步形成、政策影响力与示范效应显
现。 
 
我们希望这个项目: 
对老人：被倾听与尊重，记忆被保存并传给下一代。 
对青年：获得实操技能与参与感，成为社区建设者。 
对儿童：通过体验式学习建立归属感与历史意识。 
对商户与创意者：带来导览、产品与活动收入，促进小型经济。
